02

The Editorial Director ★

It doesn't make content. It decides what matters.

The difference between a content account and an editorial one: a content account waits for ideas; an editorial one sets the agenda. Every week the Editorial Director reads the world and decides the few conversations Leak should lead, then briefs the Content Studio to execute them.

The signals it reads

The calendar

Regulatory dates (the 1 October surcharge ban, wage and super changes), seasonal trading (footy finals, Father's Day, Christmas, EOFY), and industry moments. Timeliness is leverage.

The product

Anything Mel shipped or changed. A new feature, a better report, a fixed workflow. Every build is a potential launch theme.

The customers

Patterns across real audits. If three reports all surface supplier creep, that stops being a post and becomes a campaign. (Fed by Mel or the data.)

The feed

Gaps and momentum in what's already published: an objection that keeps coming up, a post that overperformed, a pillar that's gone quiet.

How it ranks a week

Every candidate conversation is scored on four axes, and the top five make the agenda:

  • Timeliness. Is there a reason it has to be this week? A deadline, a season, a fresh change.
  • Commercial impact. Does it move trust, bookings or the funnel, not just reach?
  • Brand fit. Is it a conversation only Leak can credibly lead?
  • Balance. Does the week span value, proof and offer, rather than three sells in a row?

What it takes in

The date, the calendar, anything Mel reports, and the state of the feed.

What it hands over

A one-page Weekly Agenda: the five conversations to lead, each with a rationale and a one-line brief, ranked. It never writes the content. It decides and briefs.

03

The Content Studio · Creative Producer mode

One input. A finished campaign.

The Content Studio is no longer a copywriter waiting for a spec. It is a producer. It never asks what format, what platform, or how many slides. It reads the four bibles, decides the campaign, and produces every asset.

It never asks

  • What format do you want?
  • Which platform?
  • How many slides?
  • Should this be a Reel or a carousel?

It decides

  • The campaign concept and hero message.
  • The best format and the platforms.
  • Every asset the idea deserves.
  • What still needs a photo or a screenshot.

The pipeline it runs, every time

  1. Read the Product Bible (what's true), the Founder Brain (the voice), the Creative OS (the look) and the Production Playbook (what works).
  2. Generate a Creative Brief: goal, audience, awareness, emotion, message, proof.
  3. Decide the campaign: the single idea, the hero message, the arc.
  4. Produce every required asset, then run the Brand Guardian checklist. If anything weakens Leak, it is rejected and improved before you see it.

05

How you run it

Two commands. That's the whole job.

Mel > Run the Content Studio: we just added supplier invoice tracking.

The Studio produces the full campaign package above, deciding every format and platform itself.

Mel > Run the Editorial Director.

It returns this week's agenda: the five conversations Leak should lead, ranked, each with a rationale and a brief ready to hand to the Studio.

The weekly rhythm

  • Monday: the Editorial Director sets the agenda for the week.
  • Through the week: the Content Studio executes each priority into finished assets.
  • Anytime something happens: tell the Studio in one sentence; it produces, the Guardian approves, you publish.

Make it automatic

The Monday agenda can run as a scheduled task, so a fresh set of priorities is waiting for you every Monday morning without you asking. That is the moment this stops feeling like a tool and starts feeling like a team.
